# ZUP LΛBS Basic Style Guide

## Core Line

Software for better decisions, cleaner execution, and real-world proof.

Supporting brand line:

ZUP LΛBS builds software that improves decisions and proves execution in the real world.

## Brand Hierarchy

- `ZUP LΛBS`: parent venture-building studio / software brand
- `SHERPΛ`: flagship decision-intelligence platform
- `MediProof`: focused healthcare execution / verification product
- `EventProof`: focused operational verification product

The brand logic is:

`decisions -> execution -> proof`

## Tone

Credible, sharp, modern, minimal.

The voice should feel:

- practical
- high-agency
- execution-first
- intelligent, but not over-explained

Copy should favour:

- clear over clever
- conviction over fluff
- reality over theory

Useful phrases:

- built for real constraints
- where guesswork fails
- useful under pressure
- if it does not hold up in the real world, it does not count

Avoid:

- theatrical hype
- macho language
- generic startup brochure wording
- words like `revolutionary`, `empower`, `cutting-edge`, `transform`

## Typography

- Display and headings: `Space Grotesk`
- Body copy and UI text: `Manrope`

Usage guidance:

- Use `Space Grotesk` for hero headlines, section headings, product names and brand-defining statements.
- Use `Manrope` for navigation, paragraphs, tags, buttons and utility text.
- Keep headlines tight and assertive.
- Use uppercase sparingly for labels, not for body paragraphs.

## Colour System

- Background: near-black charcoal
- Surface panels: layered dark greys with subtle borders
- Primary text: soft white
- Secondary text: muted grey
- Accent approach: mostly monochrome, with product logos carrying their own colour where needed

Core palette:

- `#050505`
- `#0f0f0f`
- `#151515`
- `#f2f2f2`
- `#a8a8a8`

## Layout Principles

- Strong section hierarchy over decorative complexity
- Wide negative space
- Rounded dark panels with quiet borders
- Motion should be restrained and purposeful
- Mobile layouts should stack cleanly and preserve hierarchy without feeling cramped

## Logo Usage

- Use the white ZUP LΛBS logo on dark surfaces.
- Treat SHERPΛ as the lead platform visually and strategically.
- Do not present all three ventures as equal siblings when hierarchy matters.
- Keep generous clear space around all logos.

## Lambda Usage

The lambda character `Λ` is part of the ZUP LΛBS and SHERPΛ brand language.

Recommended use:

- brand names: `ZUP LΛBS`, `SHERPΛ`
- selective high-visibility moments such as `Λustralian` or `Λustralia`

Do not force lambda substitutions everywhere. It should read as intentional branding, not as a gimmick.
